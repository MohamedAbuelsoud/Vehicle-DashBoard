Detailed Description

UART Module is for communication with the other ECU through UART. It's responsible for receiving data and updating reading values. It consists of two functions. First one (UARTvidInit) is a UART initialization function to setup the required configuration. Second one (UARTvidReceive) is a function that receives data from UART and assign it to reading.
